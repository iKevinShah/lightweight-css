About Lightweight CSS

I am a Consultant by the day and a learning web-dev by the night for some personal pet-projects. I am not that much into designing and stuff I have tried to understand various ready to use frameworks like the Bootstrap, Material Design lite, purecss, skeleton to name a few. Every time except maybe for mincss, every other one provided a lot of elements I did not need and hence had to eventually remove them (for smaller CSS size, optimization and everything around that) and maintaining it for every update that comes up (looking at you Bootstrap).So I have created a baseline css which contains only all basic elements that I am mostly going use and are grouped together so removing them is fairly easier. Making it public in case it might be of help to anyone interested.

Page Structure

The class="page" is the first class of the page, it contains all the elements except the footer. Well, this "page" class is used to keep footer at the bottom without using position absolute / fixed.

There is a class="container" which holds (contains) the elements so that they don't go out of a certain maximum width, currently set to 1346px. What this means is, for displays less than 1346px, the width will be 100%, for displays larger than that, the items within container will have width as 1346px and align themselves centrally.

We also have a class="row", which takes up all width (100%), and is used as a "row".This is also used as parent class for our grid columns

Responsive Grids

Use parent class as row i.e. class="row" and then class="col-[1-12]". Here, the number you use takes up that parts from 12 available, col-1 is 1/12th of total width and so on ... till col-12 takes up 100% of the width.
Note: This is FLEX Based. Also using col-[1-12] will stretch if rest of the row is unused.

Navbars

Use class="nav" for a navbar. 3 classes to use with navs.

  1. Use class="navbar-brand" for left sided logo, width is 50px and stretches upto 250px max
  2. Use class="nav-link" for a right sided links like the one shown below. This one auto expands, however there's a limit (of parent width - the logo).
  3. Optionally, you can use class="nav-search" for a centered search div like the one shown on this page (header above). This will stretch and take up all the available space

Clearfix

If the child divs have floats or would get floats added dynamically,A clearfix is a way for an element to automatically clear its child elements, so that you don't need to add additional markup. It's generally used in float layouts where elements are floated to be stacked horizontally. The clearfix is a way to combat the zero-height container problem for floated elements.

What is a clearfix
A clearfix is performed by using class="clearfix" to the parent div that has or would contain floated divs

Hide / Show a div

3 Classes which would help when eventually further stuff is added like Dropdowns / Toggles / etc.
Please note that there is a difference with setting the display to none vs setting visibility to hidden.

Adding class="hide" would set the div's display to none.

Adding class="show" would set the div's display to block.

Adding class="invisible" would make the div invisible.

Adding class="visible" would make the div visible, if it was not already.

Adding class="noborder" would make the element have no border.

Adding class="nopadding" will make the element have 0 padding.Example: No Padding

Adding class="nomargin" will make the element have 0 margin.

Example: This paragraph has no margin (in lightweight css, paragraph's have bottom margin unless its the last paragraph).

Adding class="nobackground" would make the element have no background. Example: No Padding

Adding class="margin20" would make the element have a 4-sided fixed 20px margin.

Final Thoughts

If you plan on using this, do note that, a lot of it is flex based so compatibility is from IE10 onwards. If you need support for older browsers, do not use this. I have left out the dropdown, tooltips, popover, modal and other things intentionally because that requires javascript to function properly and I intend on keeping this without any javascript. This is the first release, if you intend on contributing - Please do :)
